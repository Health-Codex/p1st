How to Access Admin Features

  1. Direct Access: Click the "Launch Admin Editor" button above to open the main admin interface.
  2. Provider Page Access: On the provider management page, use Ctrl + Shift + A to reveal the admin button.
  3. Alternative Access: Triple-click on the page title on the provider page to show admin controls.
  4. Security: Admin features are hidden by default to prevent patient access during normal website use.
  5. Functionality: All changes are saved locally and can be exported for backup or deployment.
