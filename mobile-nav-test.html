Mobile Navigation Test Page

This page is designed to test the mobile navigation functionality across all screen sizes.

Test Instructions

  1. Desktop Test: Resize your browser window to mobile width (<768px) to see the hamburger menu
  2. Mobile Test: Open this page on a mobile device or use browser dev tools mobile view
  3. Functionality Test: Click the hamburger menu to open/close the mobile navigation
  4. Dropdown Test: Test the "Patient Services" and "Locations" dropdown menus
  5. Accessibility Test: Use Tab key to navigate and Escape key to close menu
